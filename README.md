# Forge Fitness

A modern, premium fitness website designed with a luxury black and green aesthetic. Forge Fitness combines a cinematic landing page with an interactive workout planner, membership management, and workout tracking to deliver a professional fitness platform.

---

## Features

### Premium Landing Page
- Luxury black & lime green design
- Responsive layout
- Smooth Framer Motion animations
- Glassmorphism UI
- Modern typography
- Hero section with call-to-actions
- Animated navigation

### Membership Plans
- Cardio Membership
- Strength Membership
- Standard Membership
- Premium Membership
- Elite Personal Training Membership

### Workout Planner
Create fully customized workout routines.

Users can:

- Create multiple workout plans
- Organize exercises by weekday
- Add unlimited exercises
- Configure:
  - Weight
  - Sets
  - Repetitions
  - Rest timer
  - Exercise notes

### Workout Session Mode
Interactive workout mode includes:

- Start workout
- Complete sets
- Skip exercises
- Previous/Next exercise navigation
- Workout progress tracking
- Exercise completion status
- Session progress bar

### Progress Tracking
Track personal fitness progress including:

- Total workouts
- Training streaks
- Completed exercises
- Workout history
- Weekly activity
- Monthly activity

### Modern User Experience

- Fully responsive
- Mobile-friendly
- Smooth transitions
- Reusable React components
- Elegant animations
- Premium glass UI
- Dark luxury interface

## Installation

Clone the repository

```bash
git clone https://github.com/yourusername/forge-fitness.git
```

Navigate into the project

```bash
cd forge-fitness
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```
